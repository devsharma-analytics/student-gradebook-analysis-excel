# Student Gradebook Analysis (Excel Project)

## 📌 Project Overview
This project analyzes student academic performance using Microsoft Excel. It calculates total marks, percentages, grades, and evaluates overall performance.

## 🛠 Tools Used
- Microsoft Excel
- Logical Functions (IF)
- Percentage Calculation
- Conditional Formatting
- Charts

## 📊 Objectives
- Calculate total marks per student
- Compute percentage scores
- Assign grades based on performance criteria
- Identify top and low performers
- Visualize class performance trends

## 📁 Dataset Description
The dataset includes:
- Student Name/ID
- Subject Marks
- Total Marks
- Percentage
- Grade
- Pass/Fail Status

## 📌 Conclusion
This project demonstrates data processing, performance evaluation, and Excel-based reporting skills.
